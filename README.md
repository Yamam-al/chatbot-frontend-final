# Chatbot Frontend for the RAG-Chatbot

The Backend is found here: https://github.com/Yamam-al/school-rag-final.git 

## Setup

Look at the [Nuxt documentation](https://nuxt.com/docs/getting-started/introduction) to learn more.


1. install Node.js and mpm:
https://nodejs.org/en/download/

2. Make sure to install dependencies:

    ```bash
    # npm
    npm install

    # pnpm
    pnpm install

    # yarn
    yarn install

    # bun
    bun install
    ```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```
